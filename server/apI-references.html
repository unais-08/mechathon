Admin Management APIs

POST /admin/register Public
Register a new administrator account in the system. This endpoint creates a new admin user with the provided credentials.
{ "name": "John Doe", "email": "john@example.com", "password": "securePassword123" }
POST /admin/login Public
Authenticate an admin user and receive a JWT token for subsequent API calls.
{ "email": "john@example.com", "password": "securePassword123" }
GET /admin/ Bearer Token Required
Retrieve a list of all registered administrators. Requires valid authentication token.
GET /admin/:id Bearer Token Required
Get detailed information about a specific administrator by their unique ID.
DELETE /admin/me Bearer Token Required
Delete the currently authenticated admin account. This action is irreversible.
DELETE /admin/:id Super Admin Only
Delete a specific administrator account by ID. This endpoint requires super administrator privileges.

Blog Management APIs

GET /blogs/ Public
Retrieve all published blog posts. This endpoint is publicly accessible and returns a list of all available blogs.
GET /blogs/:id Public
Get a specific blog post by its unique identifier. Returns detailed content and metadata.
POST /blogs/ Bearer Token Required
Create a new blog post. Only authenticated administrators can publish new content.
{ "title": "How Gears Work", "content": "This is a detailed blog about gears and their mechanical applications..." }
PUT /blogs/:id Bearer Token Required
Update an existing blog post. Requires authentication and appropriate permissions.
DELETE /blogs/:id Bearer Token Required
Delete a blog post permanently. This action cannot be undone.

Sponsorship Management APIs

POST /sponsors/ Public
Submit a new sponsorship request. This endpoint allows potential sponsors to express interest and provide their details.
{ "name": "Alice Johnson", "company_name": "TechCorp Solutions", "email": "alice@techcorp.com", "phone": "1234567890", "message": "We'd love to discuss sponsorship opportunities." }
GET /sponsors/ Bearer Token Required
Retrieve all sponsorship requests. Only administrators can view the complete list of sponsor applications.
PUT /sponsors/:id/status Bearer Token Required
Update the status of a sponsorship request (e.g., approved, rejected, pending).
{ "status": "approved" }
GET /sponsors/approved Bearer Token Required
Get all approved sponsorship requests for display on public pages or internal reporting.
